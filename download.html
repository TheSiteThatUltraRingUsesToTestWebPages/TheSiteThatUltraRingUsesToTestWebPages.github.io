TIME ATTACK PREVIEW

Listed below is the download for the "Time Attack Preview". Do note that this is not a "Demo" in the normal sense. There is no intro cutscene, nor is there a boss present. You will only be able to play Final Fall as Sonic within Time Attack.

We are aiming to gather as much feedback as possible so if you can, let us know what you thought of the preview through Social Media or in the ULTRA RING COMMUNITY Discord Server! (Linked below by clicking the Discord Logo!)



Download Time Attack Preview



System Requirements:

  • OS: Microsoft 64bit Windows 10
  • Processor: 64bit Intel compatible Dual Core CPU
  • Memory: 2 GB RAM
  • Graphics: OpenGL 4-compliant onboard graphics
  • DirectX: Version 11
  • Storage: 40MB

HOW TO INSTALL

1. Click the button above to begin your download.
2. When the download is complete, extract the downloaded ZIP file and run "Sonic XG.exe" to start the game.

(ALL FILES MUST REMAIN INSIDE THE SAME FOLDER FOR THE GAME TO FUNCTION PROPERLY!)
