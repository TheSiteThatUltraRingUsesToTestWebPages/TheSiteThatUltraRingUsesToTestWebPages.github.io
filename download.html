SYSTEM REQUIREMENTS

  • OS: Microsoft 64bit Windows 10
  • Processor: 64bit Intel compatible Dual Core CPU
  • Memory: 2 GB RAM
  • Graphics: OpenGL 4-compliant onboard graphics
  • DirectX: Version 11
  • Storage: 35-50MB

HOW TO INSTALL

  1. Click the button above to begin your download.
  2. When the download is complete, extract the downloaded ZIP file and run "Sonic XG.exe" to start the game.
    (ALL FILES MUST REMAIN INSIDE THE SAME FOLDER FOR THE GAME TO FUNCTION PROPERLY!)
